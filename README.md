# Prison Architect Vietnamese Translation Mod (Language Pack)

Bản dịch tiếng Việt hoàn chỉnh 100% dành cho tựa game mô phỏng quản lý nhà tù **Prison Architect 2D** (Phiên bản Steam mới nhất, bao gồm toàn bộ các DLCs và phần chơi Campaign/Chiến dịch).

---

## 🚀 Tính năng nổi bật
* **Dịch thuật 100%**: Hoàn thiện toàn bộ hội thoại cốt truyện (Campaign), hướng dẫn điều khiển cảm ứng (Tablets), lịch sử cập nhật (Changelog) và toàn bộ nội dung của các DLCs (Island Bound, Going Green, Perfect Storm, Gangs, Undead, Future Tech Pack, Jungle Pack...).
* **Bảo vệ biến số hệ thống**: Được tối ưu hóa bằng script QA tự động, không lo lỗi crash game do sai lệch cú pháp biến (`*X`, `*Y`, `*Z`, `*Xh`, `*Ym`...).
* **Font chữ Việt hóa đồng bộ**: Sử dụng font chữ sans-serif vẽ mới đồng bộ 100% cho toàn bộ 2402 ký tự, giải quyết triệt để lỗi răng cưa, lệch dòng và đứt nét/chân chữ của các ký tự tiếng Việt có dấu.
* **Đóng gói tối giản**: Cài đặt dễ dàng chỉ bằng 1 thao tác copy-paste.

---

## 🛠️ Hướng dẫn cài đặt (Chỉ 1 Bước)

1. Tải bản mod về máy (Nút **Code** -> **Download ZIP** trên trang GitHub này).
2. Giải nén và sao chép thư mục **`VietnameseTranslation`** dán trực tiếp vào đường dẫn mod của game:
   ```text
   %localappdata%\Introversion\Prison Architect\mods\
   ```
   *(Mẹo nhanh: Nhấn tổ hợp phím `Windows + R`, dán đường dẫn `%localappdata%\Introversion\Prison Architect\mods\` vào rồi bấm Enter, thư mục chứa mod của game sẽ tự động hiện ra).*
3. Khởi động game **Prison Architect**, vào **Extras** -> **Mods**, tích chọn **Vietnamese Language Pack** của tác giả **Vu Duc Minh**, nhấn **Apply**.
4. Vào **Settings** -> **Language** và chọn **Tieng Viet** để trải nghiệm game bằng tiếng Việt hoàn chỉnh!
